# Simulador de Máquina de Turing (Proyecto 3)
Este proyecto implementa un simulador de Máquina de Turing de **una cinta**, que:

- Lee la configuración formal de la MT desde un archivo YAML.
- Simula varias entradas definidas en el mismo archivo.
- Muestra un listado de **descripciones instantáneas (IDs)** paso a paso.
- Indica si la cadena es **aceptada** o **rechazada** al finalizar la ejecución.

## Requisitos

- Python 3.10+ 
- Instalar dependencias:
